 # Running server
To run server
- Install dependencies using `npm install` or `yarn install`
- Start app using `npm run dev` or `yarn run dev`
 # Running client
To run client
- Install dependencies using `npm install` or `yarn install`
- Start app using `npm start` or `yarn start`
